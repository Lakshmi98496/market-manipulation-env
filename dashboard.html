Quick cheat sheet
🚨 ESCALATE if you see:
• Cancel rate > 35% AND strong imbalance → Spoofing
• Huge single order far from market price → Spoofing
• Many uniform-sized orders stacked → Layering
🚩 SOFT FLAG if you see:
• Same price + same size trades repeating → Wash Trading
✅ IGNORE if you see:
• No warnings in the analyst brief
• Balanced order book, low cancel rate
